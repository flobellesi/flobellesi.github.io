Que es HTML?

Actualmente la mayoria de las personas han visto cientos de páginas web pero muchos se preguntarán ¿Qué es HTML?

¿Para qué me sirve?

Si buscas en GOOGLE o en cualquier otro buscador la definicion de HTML, probablemente encontrarás está: "HTML es unlenguage de programacion que se utiliza para el desarrollo de páginas de Internet. Se trata de las siglas que corresponden a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto".

¿Para que sirve?

HTML sirve para codificar la estructura de una página web. Para ello se vale de las llamadas etiquetas o tags. Los navegadores no muestran las etiquetas pero las utilizan para representar el contenidode una página. Para saber mas...
